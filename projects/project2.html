실시간 메시징과 게시물 공유 기능이 있는 소셜 플랫폼

프로젝트 개요

이 프로젝트는 사용자 간의 연결과 상호작용을 중심으로 한 소셜 네트워크 플랫폼입니다. Firebase의 실시간 기능을 활용하여 즉각적인 알림과 메시징을 제공합니다.

주요 기능

  • 실시간 게시물 피드
  • 직접 메시징 시스템
  • 사용자 팔로우 기능
  • 좋아요 및 댓글 시스템
  • 해시태그 및 검색
  • 사용자 프로필 페이지
  • 실시간 알림
  • 이미지 업로드 및 필터

기술 스택

Vue.js 3 Firebase Realtime Database Cloud Storage Vuex Tailwind CSS WebSocket

도전 과제 및 해결책

도전: 실시간 메시징 동기화

해결: Firebase Realtime Database의 리스너를 활용하여 실시간 동기화 구현

도전: 이미지 최적화

해결: Cloud Storage의 자동 리사이징과 캐싱 활용

도전: 모바일 반응형

해결: Tailwind CSS와 Vue 반응성으로 완벽한 모바일 경험 제공
