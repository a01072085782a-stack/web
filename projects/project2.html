개발자 AI 어시스턴트

모듈식 설계와 오케스트레이션을 기반한 교육용 지능형 시스템

프로젝트 개요

초보 개발자를 위한 교육용 AI 어시스턴트입니다. 모듈식 아키텍처로 설계된 여러 개의 교육 모듈이 각각의 명확한 명세서에 따라 동작하며, 중앙의 오케스트레이터가 사용자의 질문을 분석하여 최적의 모듈들을 선택 및 조합하여 맞춤형 답변을 제공합니다.

주요 기능

  • 15+ 개의 독립적 교육 모듈
  • LLM 기반 질문 분석 및 라우팅
  • 모듈별 명확한 명세서 관리
  • 다중 모듈 조합 응답
  • 학습 진도 추적
  • 개념별 심화 학습 제공
  • 실습 문제 및 피드백
  • 학습 경로 추천

기술 스택

Python LLM API Orchestration Pattern Modular Design Node.js React MongoDB REST API

도전: 모듈 간 의존성 관리

도전: 최적의 모듈 선택 로직

해결: LLM 기반 의도 분석과 가중치 중심의 모듈 선택 알고리즘 구현

해결: 각 모듈별 명세서와 품질 체크 프로세스로 일관된 답변 보장
