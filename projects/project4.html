PWA 날씨 앱

오프라인도 사용 가능한 진보형 웹 앱

프로젝트 개요

이 프로젝트는 Progressive Web App 기술을 활용한 날씨 애플리케이션입니다. 오프라인 상태에서도 캐시된 데이터로 작동하며, 홈 화면 추가, 푸시 알림 등의 네이티브 앱 기능을 제공합니다.

주요 기능

  • 실시간 날씨 정보
  • GPS 기반 위치 인식
  • 오프라인 모드
  • 푸시 알림
  • 홈 화면 추가
  • 장기 예보
  • 다중 도시 지원
  • 다크 모드

기술 스택

Vue.js PWA Service Worker Geolocation API OpenWeatherMap API Workbox IndexedDB

도전 과제 및 해결책

도전: 오프라인 기능 구현

해결: Service Worker와 Workbox로 캐싱 전략 수립

도전: 백그라운드 동기화

해결: Background Sync API로 네트워크 복구 시 자동 동기화

도전: 성능 최적화

해결: 코드 스플리팅과 lazy 로딩으로 번들 크기 감소
