또래상담 AI 챗봇

LLM API 통합을 통한 청소년 상담 지원 플랫폼

프로젝트 개요

이 프로젝트는 청소년 또래상담을 지원하기 위한 AI 챗봇입니다. 외부 LLM API를 활용하여 사용자의 질문과 고민에 대해 감정적으로 지능적인 응답을 제공합니다. 모던한 프론트엔드 인터페이스와 안정적인 백엔드로 구성되어 있습니다.

주요 기능

  • 실시간 AI 챗 기능
  • 외부 LLM API 통합
  • 사용자 인증 및 프로필 관리
  • 대화 이력 저장 및 조회
  • 감정 상태 분석
  • 상담자 대시보드
  • 응답 품질 피드백 시스템
  • 24/7 접근 가능한 챗봇

기술 스택

React Node.js Express LLM API WebSocket MongoDB JWT REST API

도전 과제 및 해결책

도전: LLM API 비용 최적화

해결: 응답 캐싱과 프롬프트 최적화로 API 호출 횟수 감소

도전: 실시간 채팅 응답속도

해결: WebSocket을 활용한 실시간 양방향 통신 구현

도전: 사용자 프라이버시 보호

해결: 민감한 정보 정책과 데이터 암호화 적용
